## Слайд 3: Обзор системы показателей эффективности

# Обзор системы показателей эффективности

### Интегрированная система мониторинга ключевых метрик деятельности

**Основные компоненты системы:**

- **KPI администраций** — комплексные показатели эффективности работы подразделений
- **Показатели безопасности** — мониторинг лекарственной безопасности, инфекционного контроля, радиационной безопасности

**Принцип работы:**

- Автоматический сбор данных из информационных систем
- Ежедневное обновление показателей
- Цветовая индикация статуса (светофор)

## Слайд 5: Цветовая индикация показателей

# Цветовая индикация показателей

### Система светофора для быстрой оценки статуса

**Зеленый статус (норма):**
- Показатель находится в пределах установленных нормативов
- Дополнительных действий не требуется
- Пример: "Продажи за 1-й квартал: 15, статус — зеленый (норма)"

**Желтый статус (требует внимания):**
- Показатель приближается к граничным значениям
- Необходим мониторинг и профилактические меры
- Пример: "Загрузка отделения: 85%, статус — желтый (требует внимания)"

**Красный статус (проблемный):**
- Показатель выходит за установленные нормативы
- Требуются срочные корректирующие действия
- Пример: "Время ожидания приема: 45 минут, статус — красный (превышен норматив)"

**Применение цветовой индикации:**
- Автоматическое присвоение статуса на основе сравнения с нормативами
- Визуальное отображение в таблицах и отчетах
- Уведомления ответственным лицам при изменении статуса
- История изменений статусов для анализа динамики

**Доступ всех уровней к показателям с цветовой индикацией по принципу светофора**

## Слайд 7: I уровень — Детализация KPI администраций

# I уровень: KPI администраций (детализация)

### Методика расчета и текущие значения

**Формула расчета KPI администраций:**

```
KPI = (Показатель выполнения плана × 0.4) + 
      (Показатель качества × 0.3) + 
      (Показатель экономической эффективности × 0.2) + 
      (Показатель безопасности × 0.1)
```

**Текущие значения по подразделениям:**

| Подразделение | KPI | Статус | Выполнение плана | Качество | Экономика | Безопасность |
|---------------|-----|--------|------------------|----------|-----------|--------------|
| Терапевтическое отделение | 87.5 | Зеленый | 92% | 85% | 88% | 100% |
| Хирургическое отделение | 82.3 | Зеленый | 88% | 80% | 85% | 95% |
| Педиатрическое отделение | 91.2 | Зеленый | 95% | 90% | 90% | 100% |
| Реанимационное отделение | 78.9 | Желтый | 85% | 75% | 80% | 90% |
| Амбулаторное отделение | 85.6 | Зеленый | 90% | 83% | 87% | 98% |

**Интерпретация:**
- KPI > 85 — зеленый статус (норма)
- KPI 70-85 — желтый статус (требует внимания)
- KPI < 70 — красный статус (проблемный)

**Ответственный:** Главный врач, Заместитель главного врача

## Слайд 16: Методы расчета и формулы KPI

# Методы расчета и формулы KPI

### Детальное описание методик подсчета показателей

**KPI администраций:**

```
KPI = (Показатель выполнения плана × 0.4) + 
      (Показатель качества × 0.3) + 
      (Показатель экономической эффективности × 0.2) + 
      (Показатель безопасности × 0.1)
```

Где:
- Показатель выполнения плана = (Фактическое значение / Плановое значение) × 100%
- Показатель качества = Среднее арифметическое показателей качества (удовлетворенность, результаты лечения, осложнения)
- Показатель экономической эффективности = (Доходы / Затраты) × 100%
- Показатель безопасности = 100% - (Количество инцидентов / Норматив) × 100%

**Показатель загрузки отделения:**

```
Загрузка = (Количество занятых коек / Общее количество коек) × 100%
```

**Показатель оборота койки:**

```
Оборот койки = Количество выписанных пациентов / Количество коек
```

**Показатель рентабельности:**

```
Рентабельность = (Прибыль / Затраты) × 100%
```

**Показатель выполнения плана:**

```
Выполнение плана = (Фактическое значение / Плановое значение) × 100%
```

**Интерпретация результатов:**
- Все показатели сравниваются с установленными нормативами
- На основе сравнения присваивается цветовой статус
- Статусы обновляются автоматически при изменении данных
