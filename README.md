# Информационный центр и основные показатели эффективности

Интерактивное веб-приложение для мониторинга и управления показателями эффективности медицинского учреждения.

## Возможности

- **Дашборд** - обзор всех ключевых показателей
- **KPI** - детальная информация о показателях эффективности по подразделениям
- **Безопасность** - мониторинг лекарственной, инфекционной и радиационной безопасности
- **Экономика** - финансовые показатели и анализ затрат (только для уровня I)
- **Инциденты** - мониторинг и управление инцидентами
- **Настройки** - информация об уровнях доступа

## Уровни доступа

- **I уровень** - Главный врач, Заместители (полный доступ)
- **II уровень** - Заведующий отделением (данные своего отделения)
- **III уровень** - Врач, Персонал (индивидуальные показатели)

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:5173

## Сборка

```bash
npm run build
```

## Технологии

- React 18
- TypeScript
- Material-UI (MUI)
- Vite

## Структура проекта

- `src/components/` - переиспользуемые компоненты
- `src/pages/` - страницы приложения
- `src/data/` - данные и моки
- `src/types/` - TypeScript типы
