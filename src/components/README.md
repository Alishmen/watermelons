# Структура компонентов

## Организация папок

### `charts/` - Графики и визуализации
Компоненты для отображения различных графиков, диаграмм и индикаторов прогресса.

- `MonthlyProgressChart.tsx` - Кружочек с 12 секциями для отображения месяцев года
- `PieChart.tsx` - Старый компонент (для обратной совместимости)
- `utils.ts` - Утилиты для работы с данными графиков
- `index.ts` - Экспорт всех компонентов

### `ui/` - Базовые UI компоненты
Переиспользуемые UI компоненты (кнопки, инпуты, бейджи и т.д.)

### `layout/` - Компоненты макета
Компоненты для структуры страниц (Layout, Navigation, BottomNavigation и т.д.)

## Рекомендации

1. **Графики и визуализации** → `charts/`
2. **Базовые UI элементы** → `ui/`
3. **Компоненты макета** → `layout/`
4. **Специфичные компоненты страниц** → оставить в корне `components/`

## Миграция

Старые импорты:
```tsx
import { PieChart } from '../components/PieChart';
```

Новые импорты:
```tsx
import { PieChart, MonthlyProgressChart } from '../components/charts';
```
